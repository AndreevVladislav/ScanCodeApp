# ScanCodeApp

iOS-приложение для сканирования QR-кодов и штрих-кодов.

## Установка
1. Открой проект в Xcode.
2. Запусти на физическом устройстве.

## Функции
- Сканирование QR/Bar-кодов
- Отображение результата
- Сохранение результатов
